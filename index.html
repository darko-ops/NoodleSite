Reusable, living concepts that show up everywhere.

Define once. Reference anywhere. Always in sync.

Loops

Create a named handle — @project_goals, @key_decisions, @weekly_focus — that holds content you can reference from anywhere. Link any node in any note to a handle — it displays the content live. Edit the handle once, every linked node across every note updates instantly.

Echoes

Attach time-based reminders to any handle and see them on your calendar. Set a single date or build compound schedules — every 2 days and every 3 weeks. When an echo fires, it surfaces the handle and every note that references it. Your concepts don't just sit there — they come back to you on schedule.

Your @thesis_statement exists in your research note, your outline, and your draft — and when it evolves, it evolves everywhere. No copy-paste drift. No stale versions.

Thinking is nonlinear. Your tools should be too.

Track how your thinking evolves. Trust every version.

Turn thoughts into verified, immutable artifacts.

Versioning

Auto-saves daily snapshots of every modified note. Save a version manually anytime with a custom name.

Immutable

Each version is a complete snapshot — not a diff. Full note, all nodes, all media. Compressed archives with SHA-256 integrity hashes. Human-readable IDs like OBX-A1B2-C3D4.

Restore

Browse your version history, preview any state, restore to any previous point. Ideas evolve — you should be able to see exactly how.

Share

Share a live note (always current) or a frozen version (specific snapshot) via short URLs. View count tracking included.

Every thought can be traced, verified. A system of record for ideas. A provenance layer for decisions.
